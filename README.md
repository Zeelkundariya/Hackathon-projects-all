# Hackathon Projects All

This repository contains a collection of projects developed for various hackathons. Below is a detailed summary of each project, including its purpose and the technologies used.

## Project Detailed Summary

### 1. AI Interaction Interface (`AI`)
*   **Description**: A sleek, interactive chatbot interface designed for real-time communication with AI models. Features an animated UI with message bubbles, loading states, and responsive design.
*   **Tech Stack**: HTML5, CSS3 (Custom animations), JavaScript (Vanilla), SVG.

---

### 2. E-Learning Path Platform (`E-LeaningPath`)
*   **Description**: A comprehensive education portal that guides users through structured learning paths for modern web technologies. Includes separate workflows for React, Python, JavaScript, CSS, and Data Structures, complete with assignments and progress tracking.
*   **Tech Stack**: HTML5, CSS3, JavaScript.

---

### 3. Clinker Optimization System (`adani`)
*   **Description**: An advanced industrial optimization platform built for the Adani Group to manage clinker production. It performs demand uncertainty analysis, feasibility studies, and optimization runs using mathematical solvers to minimize costs and maximize efficiency across multiple plants.
*   **Tech Stack**: Python, Streamlit, MongoDB, Pyomo (Optimization), Plotly, Pandas, Gurobi/CBC Solvers.

---

### 4. AI Contract Scanner (`ai-contract-scanner`)
*   **Description**: A powerful tool designed to automate the review and analysis of legal documents. It features a browser extension to capture content and a full-stack application that uses AI to flag risks, extract key terms, and provide summaries.
*   **Tech Stack**: Next.js (React), Node.js, Express, PostgreSQL, Redis, Socket.io, OpenAI API, Tesseract.js (OCR), TailwindCSS.

---

### 5. LedgerCore - Expense Management (`expenses Management`)
*   **Description**: A refined expense tracking application featuring multi-role access (Admin, Manager, Employee). It allows for real-time expense submission, approval workflows, and data visualization of spending patterns with automated data persistence.
*   **Tech Stack**: React, Chart.js, Babel, CSS3 (Glassmorphism), LocalStorage API.

---

### 6. GitHub Portfolio Analyzer (`github-portfolio-analyzer`)
*   **Description**: An AI-driven analytics tool that evaluates developer portfolios by scanning GitHub profiles. It provides qualitative feedback on project quality, documentation, and technical depth, generating a comprehensive score and feedback report.
*   **Tech Stack**: React (Vite), Node.js, Express, Google Gemini API, SQLite, Recharts.

---

### 7. Rupiya - Agri-Fintech Platform (`rupiya`)
*   **Description**: A specialized digital ecosystem for farmers aimed at improving agricultural yields and financial inclusion. Includes modules for crop disease scanning, fertilizer advice, weather forecasting, and a marketplace for carbon credits.
*   **Tech Stack**: HTML5, CSS3, JavaScript.

---

### 8. Google Meet - Smart Meeting Experience (`Google-Meet`)
*   **Description**: A modern reimagination of Google Meet featuring a smart participant grid, real-time communication, and a comprehensive meeting analytics dashboard. It includes active speaker detection, screen sharing, and engagement metrics.
*   **Tech Stack**: React (Vite), Node.js, Express, Socket.io, MongoDB, WebRTC, Tailwind CSS, Framer Motion, Recharts.
